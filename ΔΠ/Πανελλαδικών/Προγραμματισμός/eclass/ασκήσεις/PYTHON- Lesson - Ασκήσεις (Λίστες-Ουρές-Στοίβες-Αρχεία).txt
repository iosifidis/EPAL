2. Η λίστα LOGIC περιέχει λογικές τιμές True και False. Να γραφεί συνάρτηση η οποία δέχεται τη συγκεκριμένη λίστα και, χωρίς τη χρήση αλγορίθμων ταξινόμησης, τοποθετεί στις πρώτες θέσεις της λίστας τις τιμές True και στις τελευταίες τις τιμές False

Λύση
def sort(LOGIC):
    for i in range(len(LOGIC)):
        if LOGIC[i]==True:
            LOGIC.insert(0,LOGIC.pop(i))
    return LOGIC

Αλλιώς:
def sort(LOGIC):
    pl=0
    N=len(LOGIC)
    for i in range(N):
        if LOGIC[i]==True
            pl=pl+1
    for i in range(N):
        if i<pl:
            LOGIC[i]=True
        else:
            LOGIC[i]=False
    return LOGIC
