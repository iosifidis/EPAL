Πληροφορίες σχετικά με το Πρόγραμμα Προβολής PowerPoint

Το Πρόγραμμα Προβολής Microsoft Office PowerPoint σάς επιτρέπει να προβάλλετε παρουσιάσεις που δημιουργήθηκαν με τα προγράμματα Microsoft Office PowerPoint 2000, PowerPoint XP, PowerPoint 2003 και PowerPoint 2007. Το Πρόγραμμα Προβολής PowerPoint υποστηρίζει επίσης το άνοιγμα παρουσιάσεων PowerPoint με κωδικό πρόσβασης.

ΣΗΜΕΙΩΣΗ: Για να προβάλετε παρουσιάσεις που δημιουργήθηκαν με το PowerPoint 2007, θα χρειαστεί να εγκαταστήσετε το Πακέτο συμβατότητας του Microsoft Office για τις μορφές αρχείου Word, Excel και PowerPoint του Office 2007. Δείτε τη διεύθυνση http://office.microsoft.com/ για περισσότερες πληροφορίες.

Επιστροφή στα Περιεχόμενα

Απαιτήσεις συστήματος

  • Υπολογιστής με ταχύτητα επεξεργαστή 200 MHz ή μεγαλύτερη
  • Microsoft Internet Explorer 5 ή νεότερη έκδοση
  • Υποστηριζόμενα λειτουργικά συστήματα: Microsoft Windows 2000 με Service Pack 4, Windows XP με Service Pack 1, Windows Server 2003 ή Windows Vista.
  • Μνήμη τυχαίας προσπέλασης (RAM):
    • Για Windows 2000 με Service Pack 4: 64 MB RAM
    • Για Windows XP με Service Pack 1, Windows Server 2003: 128 MB RAM
    • Για Windows Vista: 512 MB RAM
  • 5 MB διαθέσιμου χώρου στο δίσκο
  • Super VGA (800 x 600) ή υψηλότερη ανάλυση οθόνης

Επιστροφή στα Περιεχόμενα

Πολύγλωσσα θέματα

Το Πρόγραμμα Προβολής PowerPoint δεν υποστηρίζει τα Εργαλεία γλωσσικού ελέγχου του Microsoft Office.

Το Πρόγραμμα Προβολής PowerPoint δεν υποστηρίζει παρουσιάσεις αμφίδρομης γραφής που δημιουργήθηκαν με το PowerPoint 97.

Επιστροφή στα Περιεχόμενα

Εργασία με το Πρόγραμμα Προβολής PowerPoint

Το Πρόγραμμα Προβολής PowerPoint δεν εμφανίζει σωστά τις κινήσεις διαγραμμάτων. Αντί να εμφανιστούν οι κινήσεις, εμφανίζεται το διάγραμμα ολοκληρωμένο.

Το Πρόγραμμα Προβολής PowerPoint δεν υποστηρίζει το άνοιγμα παρουσιάσεων της υπηρεσίας Διαχείρισης δικαιωμάτων πληροφοριών (IRM) .

Το Πρόγραμμα Προβολής PowerPoint δεν υποστηρίζει την εκτέλεση μακροεντολών, προγραμμάτων ή το άνοιγμα συνδεδεμένων ή ενσωματωμένων αντικειμένων.

Το Πρόγραμμα Προβολής PowerPoint δεν υποστηρίζει την εμφάνιση αυτών των μορφών γραφικών, όταν συνδέονται με μια παρουσίαση: .eps, .pct, .emz, .cdr, .cgm, .wpg και .wmz.
