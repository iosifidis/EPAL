Font Sizes

Font sizes can be set using the font-size property, the value of which can be a length, percentage, or keyword.

Lengths

Absolute lengths

  • font-size: 12px
  • font-size: 12pt
  • font-size: 12q

Relative lengths

  • font-size: 1.2ex
  • font-size: 1.2em
  • font-size: 1.2vmax

Percentage

  • font-size: 50%
  • font-size: 100%
  • font-size: 150%

Keywords

  • font-size: x-small
  • font-size: medium
  • font-size: x-large
